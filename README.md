# 🌱 OptiCrop – AI Crop Recommendation System

## Overview
OptiCrop is an AI-powered Crop Recommendation System developed as part of the **SmartBridge Internship**. The application predicts the most suitable crop based on soil nutrients and environmental conditions using a Random Forest Machine Learning model. It provides recommendations through a user-friendly Flask web application.

## Features
- 🌱 Crop prediction using Machine Learning
- 🌐 Flask-based web interface
- 💾 SQLite database for prediction history
- 📱 Responsive frontend using HTML, CSS, and JavaScript

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- SQLite
- HTML
- CSS
- JavaScript

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/rockychandu/OptiCrop.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python app.py
   ```
4. Open your browser and visit:
   ```
   http://127.0.0.1:5000
   ```
